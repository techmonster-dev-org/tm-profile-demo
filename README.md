## How This Profile Is Verified

Each project is reviewed by the TechMonster AI Coach, which:

1. **Reads the actual submitted code** against a project-specific rubric
2. **Checks genuine understanding** — the student must explain every non-trivial decision
   in their own words (journal entries). Code that works but wasn't understood is flagged,
   not marked verified.
3. **Records a commit SHA anchor** — each review is pinned to the exact Git commit it
   evaluated, creating a cryptographic link to the private work.

The per-track detail pages (linked above) show every project score, review date, and a link
to the original feedback. The verified-record.json files in each track folder contain machine-readable proof.
